# 爆炸保护插件 / Explosion Protector Plugin
## 已测试版本 / Tested Version
    Paper 1.18.1
## 功能 / Features
防止爆炸破坏方块 / Prevent explosion destroying block  
防止爆炸破坏以下实体: / Prevent explosion destroying the following entities:
 * 画 / Painting
 * 盔甲架 / Armor Stand
 * 物品展示框 / Item Frame
 * 荧光物品展示框 / Glow Item Frame
